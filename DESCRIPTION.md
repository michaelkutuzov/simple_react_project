Страница, отображающая список карточек работников в виде сетки. Каждая карточка показывает картинку, имя, фамилию, должность работника. При клике на карточку открывается на всю страницу компонент карточки. Отображает картинку, имя, фамилию, должность работник. В хедере страницы есть кнопка, которая позволяет добавить нового работника. При её нажатии открывается компонент формы, где можно ввести имя, фамилию, должность работника и добавить ссылку на картинку. На форме есть кнопка Сохранить. При нажатии кнопки Сохранить новая карточка отображается рядом с существующими. Также на каждой карточке работника есть кнопка Удалить. Удалённая карточка работника перестаёт отображаться в списке карточек работников
